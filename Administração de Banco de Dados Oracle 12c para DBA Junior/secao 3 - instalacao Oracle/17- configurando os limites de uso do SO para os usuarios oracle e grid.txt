17- configurando os limites de uso do SO para os usuarios oracle e grid



o oracle faz varias checagens para ver esses limites 


vim /etc/security/limits.conf



alocação por usuario 
soft - minimo 
hard - maximo 


numero de arquivos que podem ser abertos simitaneamente 



# CONFIGURACOES DO ORACLE
oracle  soft    nproc   2047
oracle  hard    nproc   16384
oracle  soft    nofile  1024
oracle  hard    nofile  65536
oracle  soft    stack   10240
oracle  hard    stack   32768
grid    soft    nproc   2047
grid    hard    nproc   16384
grid    soft    nofile  1024
grid    hard    nofile  65536
grid    soft    stack   10240
grid    hard    stack   32768
# FIM DAS CONFIGURACOES DO ORACLE
